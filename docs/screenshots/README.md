# Screenshots

Add your screenshots here after running the project locally.

## Required screenshots for the README:

| Filename | What to capture |
|---|---|
| `fake_prediction.png` | Fill the "Fake Example" quick-fill button → Submit → show the result panel with red verdict |
| `real_prediction.png` | Fill the "Real Example" quick-fill button → Submit → show the result panel with green verdict |
| `model_comparison.png` | After a prediction, click "Compare All 3 Models" → show the comparison table |
| `dashboard.png` | Switch to Dashboard tab after making a few predictions |
| `model_info.png` | Switch to Model Info tab → shows accuracy table + feature importance bars |

## How to take a good screenshot:
1. Run `python manage.py runserver`
2. Open http://127.0.0.1:8000/
3. Use browser full-screen mode (F11) for clean screenshots
4. On Mac: Cmd+Shift+4 to select area
5. On Windows: Win+Shift+S to select area
